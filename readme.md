**[Formula 1 Telemetry Analysis]**
* **Goal:** Visualize driver performance characteristics using real-time telemetry data.
* **Tech:** Python, FastF1 API, Matplotlib (LineCollection), NumPy.
* **Details:** Analyzed Lewis Hamilton's fastest lap at the Mexico Grand Prix. Created a geospatial track visualization that maps gear usage and braking zones onto the track layout using color-coded segments.
